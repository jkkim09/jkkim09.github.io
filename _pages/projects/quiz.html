3. KaKao 2020 문제 정답률: 34.4%
정답코드 https://github.com/jkkim09/quiz-javascript/tree/master/quiz-4
예문이 길어사 잠깐 이해하는데 버퍼링이 있었다. 하지만 간단하게 생각하기로 했다 검색할 단어는 소문자로 이루어져있고 queries 에서 ? 표는 a~z 까지의 단어를 의미한다. 딱 요정도만 생각하고 코딩을 식작했다 그리고 바로 생각한것이 정규식 이였다. 그래서 ? 를 정규식 [a-z] 까지로 치환하고 단어의 길이도 고려해 정규식 끝을 의미하는 $ 문자까지 포함한 정규식을 만들어 javascript test 함수를 이용하여 단어를 확인하고 true 일경우 결과값 index에 +1 을하여 return 하였다.

위에서 말한 방법으로 하면 일단 동작은된다 하지만 리소스 측면에서는 늦을거 같다는 생각이들어 인터넷에서 검색을 해보니 정규식으로 했을때와 Trie 구조로 했을떄의 차이가 많이난다는것을 알게 되었다 차 후 Trie 구조를 이용하여 다시 풀어볼 예정이다.
